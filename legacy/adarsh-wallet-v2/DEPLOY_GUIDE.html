Set Mumbai IST Timezone (UTC+5:30)
Ensures all dates/times show correctly for India
🕐 All expense dates and timestamps will use Asia/Kolkata (IST UTC+5:30) — Mumbai timezone

In src/utils/constants.js, ensure CONFIG has:

constants.js
export const CONFIG = {
  // ... your API keys ...
  TIMEZONE: 'Asia/Kolkata',   // IST Mumbai
  TIMEZONE_OFFSET: '+05:30',
  DATE_LOCALE: 'en-IN',
};

In Code.gs (Apps Script), update formatDate():

Code.gs — formatDate()
function formatDate(d) {
  if (!d) return '';
  if (d instanceof Date)
    return Utilities.formatDate(d, 'Asia/Kolkata', 'yyyy-MM-dd');
  return String(d).substring(0, 10);
}
